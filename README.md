# Pokedex
Pokedex App that list pokemons using React and Material-UI. You can also search for a pokemon on the Search Bar. 

Pokemon Data was from PokeApi, a RESTful API.

# Dependencies
- Material-UI 
- ReactJS
- React-Router
- React-Router-Dom
- Axios
